mRedis
======

a multi core implement of Redis (http://redis.io)

building
===
build only

    ./build.sh

build and run

    ./build.sh r 
    
or

    ./build.sh run

build and run with parameter

    ./build.sh run -p 6379

running
===

    ./mRedis -p 6379
    
client
===
completely compatible with redis

see more at:http://redis.io/clients
